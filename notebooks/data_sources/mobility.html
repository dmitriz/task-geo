This notebooks show how to use the mobility data source which extracts data from the Google Mobility reports.

To extract the information, this data source needs to download the PDFs, altough this operation only is required to be performed one time. To use the data source we just need to load it and pass it the path where we want to download the PDF files, we donâ€™t have to worry about the path not existing, as it will create it for us.

Now we will start by importing the data_source loader and get the mobility data source

[1]:

from task_geo.data_sources import get_data_source

data_source = get_data_source('mobility')

Now we just need to call it:

[2]:

data = data_source()
